# Seed Data

이 폴더는 RAG 학습용 샘플 문서를 보관하는 곳입니다.

## wellness_docs.json

- 이 파일에는 심리 및 웰빙 교육용 문서가 들어갑니다.
- 각 문서는 `title` 과 `content` 값을 가집니다.
- 이 문서들은 나중에 데이터베이스에 저장되고, 임베딩 생성 후 검색에 사용됩니다.
- 내용은 의료 진단이나 치료 목적이 아니라, 일반적인 정보 제공과 학습 목적에 맞춰 작성합니다.

## 예시 주제

- 스트레스 이해하기
- 기본 수면 위생
- 불안할 때 호흡 조절
- 감정 기록하기
- 자기비판 줄이기
